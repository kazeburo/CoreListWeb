## Web interface for Module::CoreList

Web interface for Module::CoreList. Corelist-Web will automatically update to the latest version of Module::Corelist.

This is based on tokuhirom's https://github.com/tokuhirom/corelist-web

## WebSite

https://corelist.nomadscafe.jp/

## How to start

Use Docker

   $ docker pull ghcr.io/kazeburo/corelistweb:latest

or 

   $ cd /path/to
   $ carton install
   $ carton exec -- ./server.pl
